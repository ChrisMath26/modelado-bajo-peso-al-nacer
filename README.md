# Modelado bajo peso al nacer

Proyecto de análisis y modelado predictivo orientado al estudio del bajo peso al nacer.

## Objetivo

Construir un flujo reproducible de análisis de datos que incluya:

- carga y limpieza de datos,
- análisis descriptivo,
- análisis exploratorio,
- visualización,
- análisis estadístico,
- modelado predictivo,
- documentación del uso de IA.
